Metronic Admin (Framework) Features

In the frontend theme you can use nearly all the features from Metronic Admin Theme. First you will just need to include 2 below global css files:

../../assets/global/css/components.css
../../assets/global/css/plugins.css

And selected plugin's css and js files accordingly.
